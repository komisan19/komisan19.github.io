A Happy Birthday!

この問題はシンプルに標準入力を使えば解ける問題です。 Goの標準入力はfmt.Scan()を使いことが多いと思いますが、今回は練習も兼ねてbufioを使ってみました。 fmt.Scan()を3×10^5以上行うとTLEになるそうなので、今後はbufioを使っていこうと思います。

bufioを使った書き方はこちらです。

package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    r := bufio.NewReader(os.Stdin)

    var a, b int
    fmt.Fscan(r, &a, &b)
    ...

}
